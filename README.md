# Knowledge-Farm-Android

# 项目简介

> ​		知识农场是一款面向一到三年级的小学生群体的养成类游戏。通过游戏的方式来促进孩子学习与巩固知识，降低了孩子对学习的一种抵触心理，进而减少家长来源于孩子学习的一种压力，旨在让学习知识成为轻松游戏的额外赠品。

# 技术栈

> 客户端：
>
> ​		图形框架：Glide
>
> ​		事件框架：EventBus
>
> ​		网络请求框架：okHttp
>
> ​        服务器系统：Ubuntu 18.04
>
> ​        后台开发框架：JFinal    ===>**SpringBoot+JPA**
>
> ​        数据库：mysql===>**msql+druid**
>
> ​		**日志框架：slf4j+logback**

# 具体流程和功能

# **客户端**

## 注册方式

- QQ登录->完善信息
- 账号注册

## 登录方式

- QQ登录
- 账号登录

> 首次登录后，再次登录时
>
> - QQ登录会自动登录
> - 账号登录会自动完善账号信息，如果上次点了记住密码还会完善密码信息

> 忘记密码 ，绑定邮箱后，可以通过邮箱找回

## *[后台管理系统](http://39.106.18.238:8081/)-用户管理*

- *可以修改用户的信息*
- *可以删除/恢复用户（封号/解封）*

## 新手引导

- 用户第一次进入主界面和商店会有出现新手引导界面

## 每日任务

1. 每日签到或者完成种植、浇水等任务
2. 完成后可以领取相应的奖励

## 种植作物

1. 首先去**商店**购买喜欢的种子
2. 选择想种的土地
3. 点击土地，在弹出的**背包**里选择要种的种子
4. 种植完成

> 背包中可以查看拥有的种子

## 商店

1. 进入商店
2. 切换商品标签
3. 选择要购买的作物、宠物或者道具购买

## *后台管理系统-作物管理*

- *可以上架/下架作物*
- *修改作物信息*

## *后台管理系统-宠物管理*

- *上架/下架宠物*
- *修改宠物信息*
- *修改**宠物的功能***
  - **不同宠物有不同的功能**
  - **相同宠物不同成长阶段功能效果不同**

## *后台管理系统-宠物道具管理*

- *上架/下架宠物道具*
- *修改宠物道具信息*

## 浇灌作物

1. 首先保证有足够的水，如果没水了需去**学习中心**答题获得
2. 选中下方工具里的**水壶**
3. 点击需要浇水的土地
4. 浇灌完成，作物成长值增加

> 需注意：植物会自然生长，但会随机发生干旱，这时无法进行施肥，需要浇水才能继续生长，所以虽然可以无限浇水，但是水用完了植物一旦干旱就不会自然生长了

## 作物施肥

1. 首先保证有足够的肥料，如果没肥料了需去**学习中心**答题获得
2. 选中下方工具里的**肥料**
3. 点击需要施肥的土地
4. 施肥完成，作物成长值增加

## 学习中心

1. 点击主页面学习中心进入
2. 选择对应科目进行答题

### 科目：

- 数学

- 英语

- 语文

### 题型：

- 单选题
- 填空题
- 判断题

> 题目难度随用户设置的年级改变

## *后台管理系统-题目管理*

- *添加、修改题目*
- *批量添加修改题目*
- *导出题目数据*

## 收获作物

1. 选中下方工具里的**手**
2. 点击成熟的作物
3. 完成收获，获得金币和经验

## 扩建土地

1. 点击土地上的**扩建**标志
2. 支付相应金币
3. 完成扩建

## *后台管理系统-土地管理*

- *可以查看/修改用户土地状态*

## 添加好友

1. 点击**我的好友**
2. 选中  ’推荐好友‘
3. 直接搜索账号或添加推荐的人
4. 点击添加按钮发送申请

## 删除好友

1. 点击**我的好友**
2. 直接搜索账号或直接找到要删除的人
3. 点击删除按钮删除

## 同意/拒绝他人好友申请

1. 点击**通知**图标
2. 选择好友申请
3. 点击同意或拒绝

## 访问好友农场

1. 点击我的好友
2. 点击或搜索要访问的好友
3. 点击**头像**进入对方的农场
4. 可以帮好友浇水或施肥可以获得额外金币奖励

## 个人信息

1. 点击**自己头像**
2. **个人信息页**可以查看账号或者进行昵称修改、年级修改、密码修改、头像修改、绑定或解绑QQ和邮箱
3. 点击**系统设置**可以**切换账号、(手动)检查更新**

## *后台管理系统-版本更新管理*

- *点击添加*
- *填写本次更新内容*
- *上传apk文件*

## 通知系统

1. 系统通知
2. 好友申请通知
3. 发送申请通知
4. 普通通知（好友互动相关的通知）

## *后台管理系统-通知管理*

- *可以发送推送（全部人、部分人） 手机状态栏上的通知用于提醒更新等*
- *可以发送系统通知（全部人、部分人）*

## 宠物仓库

1. 点击**宠物**图标
2. 点击左侧宠物列表切换要查看的宠物
3. 点击更换按钮切换主界面展示和用于对战的宠物
4. 点击下方**道具**喂养选中的宠物

## 好友宠物PK

1. 确保正在使用宠物体力值充足
2. 点击好友列表对应好友pk按钮，消耗体力值开始pk
3. 准备就绪，触摸开pk
4. 答题决定本回合是否有攻击资格
5. 直到一方血量归零pk结束，中途离开判负
6. pk胜利宠物智力值会提升，达到一定程度宠物会进入下一成长阶段

## *后台管理系统-用户宠物管理*

- *可以修改用户宠物的各项属性*
- *可以切换正在使用的宠物（未使用—>使用）*

## *后台管理系统-管理员管理*

- *可以添加新的管理员*
- *可以删除和恢复管理员*
