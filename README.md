# Knowledge-Farm-Android

# 项目简介

> ​		知识农场是一款面向一到三年级的小学生群体的养成类游戏。通过游戏的方式来促进孩子学习与巩固知识，降低了孩子对学习的一种抵触心理，进而减少家长来源于孩子学习的一种压力，旨在让学习知识成为轻松游戏的额外赠品。

# 技术栈

> 客户端：
>
> ​		图形框架：Glide
>
> ​		事件框架：EventBus
>
> ​		网络请求框架：okHttp

>

>
> ​        数据库：mysql===>**msql+druid**

> ​		**日志框架：slf4j+logback**

# 具体流程和功能

# **客户端**

## 注册方式

- QQ登录->完善信息
- 账号注册

## 登录方式

- QQ登录
- 账号登录

> 首次登录后，再次登录时
>
> - QQ登录会自动登录
> - 账号登录会自动完善账号信息，如果上次点了记住密码还会完善密码信息

> 忘记账号密码 ，通过邮箱找回

## 种植作物

1. 首先去商店购买喜欢的种子
2. 选择想种的土地
3. 点击土地，在弹出的背包里选择要种的种子
4. 种植完成

> 背包中可以查看拥有的种子

## 浇灌作物

1. 首先保证有足够的水，如果没水了需去学习中心答题获得
2. 选中下方工具里的水壶
3. 点击需要浇水的土地
4. 浇灌完成，作物成长值增加

> 需注意：植物会自然生长，但会随机发生干旱，这时无法进行施肥，需要浇水才能继续生长，所以虽然可以无限浇水，但是水用完了植物就不会自然生长了

## 作物施肥

1. 同上

## 收获作物

1. 选中下方工具里的手
2. 点击成熟的作物
3. 完成收获，获得金币和经验

## 扩建土地

1. 点击土地上的扩建标志
2. 支付相应金币
3. 完成扩建

## 添加好友

1. **点击我的好友**
2. **选中  ’推荐好友‘**
3. **直接搜索账号或添加推荐的人**
4. **点击添加按钮发送申请**

## 删除好友

1. **点击我的好友**
2. **直接搜索账号或直接找到要删除的人**
3. **点击删除按钮删除**

## 访问好友农场

1. 点击我的好友
2. 点击或搜索要访问的好友
3. 进入对方的农场
4. 可以帮好友浇水或施肥可以获得额外金币奖励

## 同意/拒绝他人好友申请

1. **点击通知图标**
2. **选择’好友申请‘**
3. **点击同意或拒绝**

## 游戏设置

1. **点击自己头像**
2. 可以进行昵称修改、年级修改、密码修改、头像修改、绑定或解绑QQ和邮箱、注销登录

## 学习中心

1. 数学
2. 英语
3. 语文

> 题目难度随用户设置的年级改变

## 通知系统

1. **系统通知**
2. **好友申请通知**
3. **发送申请通知**
4. **普通通知**

## 每日任务

1. **签到或者完成种植、浇水等任务**
2. **可以领取相应的奖励**

# [后台管理系统](http://39.106.18.238:8081/)

## 用户管理

- 可以修改用户的信息
- 可以删除用户（封号）

## 作物管理

- 可以上架作物
- 修改作物信息

## 管理员管理

- 可以添加新的管理员
- 可以删除和恢复管理员

## 土地管理

- 可以修改用户土地状态

## 宠物管理

- **上架/下架宠物**
- **修改宠物信息**
- **上架/下架宠物道具**
- **修改宠物道具信息**

## 通知管理

- **可以发送推送（全部人、部分人） *手机状态栏上的通知用于提醒更新等***
- **可以发送系统通知（全部人、部分人）**
